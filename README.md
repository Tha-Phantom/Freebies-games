Requirements
- Node.js 18+ (or Node 16+)
- npm

Getting started
1. Install dependencies:
   npm install
2. Run the dev server:
   npm run dev
3. Open the shown URL

Build
- Production build:
  npm run build
- Preview production build:
  npm run preview

- Host as server in the same web:
  npm run dev -- --host 2>&1


API
- This app uses the public GamerPower JSON API:
  https://www.gamerpower.com/api/giveaways
